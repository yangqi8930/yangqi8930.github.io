python 绘图总结

导入matplotlib包后，可以设置如下代码，解决mac电脑中文显示乱码的问题。

饼图

绘图语句：
plt.ple(x, explode, labels, colors, autopct=, pctdistance, shadow, startangle, radius, wedgeprops, textprops, center, )
参数详解：
labels：标签
colors：颜色
autopct：百分比
pctdistance：百分比标签与圆心距离
shadow：是否添加饼图阴影效果
labeldistance：设置各扇形标签与圆心距离
startangle：设置饼图的初始摆放角度
radius：设置饼图半径大小
counterclock：是否逆时针呈现
wedgeprops：设置饼图内外边界的属性
textprops：设置饼图中文本属性
center：设置中心位置

条形图

plt.bar(x, y, width, bottom, color, linewidth, tick_label, align)
x：指定x轴上数值
y：指定y轴上数值
width：指定条形图宽度
color：条形图的填充色
edge：条形图的边框色
linewidth：条形图边框宽度
tick_label：条形图的刻度标签
align：指定x轴上的对齐方式

直方图

plt.hist(x, bins, range, normed, cumulative, bottom, align, rwidth, color, edgecolor, label)
x：数据
bins：条形个数
range：上下界
normed：是否将频数转换为频率
cumulative：是否计算累计频率
bottom：为直方图的每个条形添加基准线，默认为0
align：对齐方式
rwidth：条形的d宽度
color：填充色
edgecolor：边框颜色
label：标签

散点图

plt.scatter(x, y, s, c, maker, cmap, norm, alpha, linewidths, edgecolor, label)
x：x数据
y：y轴数据
s：散点大小
c：散点颜色
maker：散点图形状
cmap：指定某个colormap值，该参数一般不用，用默认值
alpha：散点的透明度
linewidth：散点边界线的宽度
edgecolors：设置散点边界线的颜色
