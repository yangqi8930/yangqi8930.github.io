Hexo+github 部署搭建个人博客(IOS系统)

准备工作

  1. 电脑（Mac）
  2. 电脑上安装Node.js和git（由于Mac电脑自带Git，因此无需另外w安装。Node.js下载地址)
  3. 检查Node.js和Git是否安装成功： 出现版本号即为安装成功

    安装Hexo

    $ npm install -g hexo-cli

    建站

    配置文件修改

    配置根目录_config.yml有关deploy的部分：

    安装deploy插件

    npm install hexo-deployer-git --save

    配置Github

  4. 创建一个自己的github账号
  5. 新建一个仓库：仓库名设置为：username.github.io
  6. 配置ssh：ssh-keygen -t rsa -C “邮件地址”‘’’。连续三次回车，然后打开用户目录，找到.ssh\id_rsa.pub文件，将里面的密码复制，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：将刚复制的内容粘贴到key那里，title随便填，保存。

    使用Hexo写博客

    hexo init test # 新建一个test文件夹，生成一些必须的静态文件
    hexo g #生成网站文件
    hexo s #启动服务
    hexo d #上传到github
    然后打开网址：SuperMario的博客就可以看到自己的博客啦
