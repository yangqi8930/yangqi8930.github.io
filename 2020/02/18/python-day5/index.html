13. 生成满足均匀分布的坐标点

random 模块，uniform(a,b) 生成 [a,b)内符合平均分布的一个随机数。

使用 PyEcharts 绘图，版本 1.6.2。
使用前要先导入pyecharts包，可以通过pycharm–>preference–>project–>project interpreter 点击加号，搜索pycharts 然后点击install即可安装成功。
注意，运行以下代码至少保证版本要在 1.0 以上：
from pyecharts.charts import Scatter
import pyecharts.options as opts
from random import uniform

def draw_uniform_points():
x, y = [i for i in range(100)], [
round(uniform(0, 10), 2) for _ in range(100)]
print(y)
c = (
Scatter()
.add_xaxis(x)
.add_yaxis(‘y’, y)
)
c.render()

draw_uniform_points()
