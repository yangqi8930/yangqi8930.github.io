Python之数学运算、逻辑运算和进制转化相关的16个内置函数

学习之前需要先了解几个概念：位置参数，关键字参数等几个概念：

  1. 位置参数： 如上函数funcA中，定义了两个参数a和b ，当调用该函数时，传入了两个值，2和3，那么按照参数顺序和个数保持一致的原则，2赋值给a，3复制给b，所以结果时a=2，b=3

重点：位置参数的参数顺序和个数需和函数定义的保持一致
2. 关键字参数：
还是刚刚的函数funcA(a,b)，如果我们在调用的时候，明确指定参数值赋给哪个形参，例如：
funcA(a=100,b=99)
和
funcA(b=99,a=100 )
得到的结果是一样的。
也就是说，*关键字参数可以让我们在调用函数的时候打乱参数传递的顺序。
也可以通过如下方式调用：
funcA(100,b=99)
但是这种调用方式时，位置参数一定要在前**
关于函数原型的学习：
例如：
max(iterable,[,key,default])
函数行参列表中，符号 * 表示，后面的形参只能为关键字参数,也就是说max参数用法为：


再如 sum函数：
sum(iterable,/,start=0)
/ 表示该符号前面只能是位置参数

而以下调用时非法的，iterable参数不能被赋值为关键字实参：
sum(interable=a,start=2)

数学运算

1. len(s):返回对象内元素的个数

dic = {‘a’:1,’b’:3}
print(len(dic))

2. max(iterable,*[,key,default])

print(max(21,2,4,2))

3. pow(x,y,z=None,/)：x为底的y次幂（次方）,如果z给出，取余。

print(pow(3,2,4))

4. round(number[,ndigits])：四舍五入，ndigits代表小数点吼保留几位

print(round(4/3,3))

5. sum(iterable,/,start=0)：求和

a = [1,2,1,4,1]
print(sum(a))
print(sum(a,10))

6. abs(x,/):求绝对值

print(abs(-6))

7. divmod(a,b)：分别取商和余数

print(divmod(10,3))

8. complex([real[,imag]])创建一个复数

print(complex(1,2))

9. hash(object)：返回对象的哈希值

print(hash(‘xiaoming’))# 可哈希的对象可以为字符串或者数值，但不能直接应用于list，set，dict
哈希的概念有待理解

10. id(object):返回对象的内存地址

print(id(‘xiaoming’))

逻辑运算

1. all(iterable)：

接受一个迭代器，如果迭代器中的所有元素都为真，返回True，否则返回False：
print(all([1,2,3,1]))

2. any(iterable)：

接受一个迭代器，如果迭代器中的有一个元素都为真，返回True，否则返回False
print(any([0,0,0,[]]))# 0，空和false都算Flase
print(any([0,0,1]))

进制转化

1. ascii(object)：调用对象的repr()方法，获得该方法的返回值。

2. bin(x)：将十进制转换为二进制

print(bin(10))

3. oct(x)：将十进制转换为八进制

print(oct(9))

4. hex(x)：将十进制转换为十六进制

print(hex(15))
