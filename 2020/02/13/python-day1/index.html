Python 数据类型总结（一）

数值型

  1. int: 整型（eg : 0，1，-1）
  2. float: 浮点型（eg : 1.2，3.0，-0.1）
  3. bool: 逻辑对象（eg : true ,false）

容器型：

  1. list 型：

创建：使用一对中括号[ ]，创建list对象：list = [1,3,5] #list 对象
访问：list对象为一个有序集合对象，使用索引进行访问，例如：L[0] 代表访问list的第一个元素；也可以使用倒序访问，例如：L[-1] 代表倒数第一个元素，依次类推。
添加：
list.append() #追加新元素到list末尾;
list.inser(location,value) #location 代表插入的位置，value代表要插入的值，
例如：
list.insert(1,’Mario’)将’Mario’插入list第二个位置，原来第二个位置的元素依次向后移动一位。
删除：
list.pop() #删掉list的最后一个元素
或者已知一个元素的位置：list.pop(1) #删掉索引为1的元素

  1. tuple 型：

创建：使用一对括号( )，创建一个tuple对象：tup = (1,3,5) #tuple 对象。注意：单元素tuple的创建方法为：t=(1,)，多加一个逗号’,’。
访问：与list一样：t[0],t[-1]。注意：tuple一旦创建完成，不可更改。
